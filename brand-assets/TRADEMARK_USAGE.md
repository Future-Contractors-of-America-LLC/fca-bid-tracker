# FCA & Auricrux Trademark Usage Guidelines

**For internal use and partner licensing. Not legal advice.**

## Owner

Future Contractors of America LLC

## Registered / pending marks

| Mark | Use |
|------|-----|
| FCA Hex Mark | App icon, favicon, product chrome |
| FCA / Future Contractors of America | Word marks with hex or standalone in headers |
| Auricrux Crux Mark | Intelligence layer, dock, academy/support accents |
| Auricrux Crux Pulse | Loading states for Auricrux layer only |
| FCA Contractor Command | Product name in SaaS and mobile |

## Rules

1. **Do not** alter hex or crux geometry, colors, or proportions.
2. **Do not** use Auricrux gold on primary FCA navigation (gold is Auricrux-only).
3. **Do not** use Tyler or third-party marks on customer routes.
4. **Always** show TM notice on first prominent use per page: see `/ip`.
5. **Minimum clear space:** 1x icon height around hex and crux marks.

## File locations

- Source: `brand-assets/`
- Live specimens: `/brand/` on website
- React: `src/components/FcaBrandMark.jsx`, `AuricruxBrandMark.jsx`

## Animation

Only **Auricrux Crux Pulse** is approved for motion trademark. Do not animate the FCA hex mark.
