# 062A Package Alignment

## Principle
Commercial packaging must map to actual reachable product surfaces.

## Minimum package truth
- SaaS entitlement should map to reachable portal routes
- LMS entitlement should map to academy routes
- Auricrux entitlement should map to `/portal/auricrux` and supporting comms language
- Comms entitlement should align with enabled channels shown in profile/admin surfaces

## Drift risk
If a pricing or profile package implies capabilities not surfaced by route access, trust and revenue readiness degrade.
