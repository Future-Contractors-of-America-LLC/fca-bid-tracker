# FCA Packets

Canonical location for sequenced FCA / Auricrux implementation packets, validation packets, remediation packets, persistence packets, and related ordered execution artifacts.

## Placement rule

New sequenced packet artifacts should be saved here by default:

```text
docs/packets/
```

## Naming rule

Use canonical names such as:

```text
FCA_PACKET_049N_SAAS_LMS_DURABLE_TARGET_01_APPLY_PACKET.md
```

## Operating rule

If a meaningful packet artifact is generated during a run and repository tools are available, it should be saved here during that same run rather than left only in chat output.
