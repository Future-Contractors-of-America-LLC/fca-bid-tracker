# Module 27 - Auricrux Autonomous Talent Orchestrator

## Operating Model
Module 27 is no longer an HR workflow tracker. It is an autonomous talent orchestration surface where Auricrux provisions human capacity against operational objectives.

## Core Autonomy Functions
- Autonomous sourcing and pre-screening pipeline.
- Skill-gap predictive provisioning from scheduling horizon.
- Zero-touch interview gating with 95%+ Go threshold.
- Credential-gated entry with FCA Core Certification requirement.
- Autonomous onboarding: credential provisioning + project assignment + first task mapping.

## Culture Weighting
Culture is explicit and configurable:
- Safety ownership.
- Accountability.
- Team humility.

Weights are operator-controlled and used in candidate culture alignment scoring.

## Workforce Stewardship
Stewardship logic tracks:
- Under-utilization and upskilling opportunities.
- Burnout risk and retention intervention cues.
- Internal mobility and capability promotion pathways.

## Human Escalation Protocol
Auricrux escalates sensitive cases with a contextual dossier when triggered by:
- Performance disciplinary actions.
- Complex salary negotiations.
- Culture-fit disputes.

Dossier output includes scenario summary, data points, and three recommended paths for executive decision.

## Validation Gate
`npm run validate:module27-talent-autonomy` verifies:
- Auricrux-only mode and zero-human baseline.
- Credential gate behavior.
- Culture scoring availability.
- Zero-touch interview eligibility path.
- Predictive provisioning signal path.
- Founder escalation and dossier generation.
